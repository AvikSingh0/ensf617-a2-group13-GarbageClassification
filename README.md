## 1. Introduction

The city of Calgary required a system that would classify household garbage into one of the following four categories: Green, Blue, Black, and Other using a cellphone image and a short sentence description. The system will use a multi-modal classification system to accurately classify waste that is related to Calgary's “what goes where” waste management guidelines.
